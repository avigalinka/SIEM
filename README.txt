SIEM project by Avigail Rachlis
===============================
SIEM (Security Information &amp; Event Management) – software product that provides a real-time
analysis of security alerts generated by applications and network hardware.




Files
-----

- DB_Check_Conn.py - connection to DataBase
- LogAnalyzer.py - log analizing file that detect Suspicious port, Port Scan, Ping Sweep, Ping Sweep with time aspect
- parser.py - read logs from file and send them to DataBase
- sniffer.py - sniff network or firewall, collect logs and write them in log files in format, that readable by parser.py file


Requirements
------------
1. Ubuntu Server with MySQL Database
2. Windows computer
3. Kali Linux - attacker
